# open-source-contribution-skill

Public home for the **`open-source-contribution`** skill used by ByteClaw.

This OpenClaw skill helps an agent contribute to GitHub projects like a careful open-source contributor:

- read the issue first
- keep scope tight
- fork when appropriate
- create a dedicated branch
- commit cleanly
- open a pull request with issue references and a clear summary

## Contents

- `open-source-contribution/` → editable skill source
- `open-source-contribution.skill` → packaged skill artifact

## Skill summary

The skill is designed for workflows like:
- pick up a GitHub issue
- inspect repo + issue context
- implement only the requested change
- push to a fork branch
- open a clean PR
