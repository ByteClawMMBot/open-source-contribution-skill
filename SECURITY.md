# Security Policy

## Reporting a vulnerability

If you believe this repository contains a security issue, please report it privately rather than opening a public issue first.

For low-risk documentation or packaging mistakes, a normal issue is fine.
For anything that could affect user safety, credentials, automation security, or supply-chain trust, prefer private disclosure.

## Scope

This repository mainly contains an OpenClaw skill and its packaged artifact. Relevant security concerns may include:

- unsafe contribution guidance
- instructions that encourage risky repository operations
- accidental leakage of secrets or credentials
- packaging artifacts that do not match source
- misleading or dangerous defaults

## Response expectations

Reports will be reviewed and triaged as quickly as practical.
